# Program Pengelolaan Nilai Mahasiswa

## Penjelasan Konsep Array

Array adalah struktur data yang digunakan untuk menyimpan sekumpulan nilai dalam satu variabel dengan tipe data yang sama.

Pada program ini, array `nilai` digunakan untuk menyimpan 10 nilai mahasiswa yang diinput oleh pengguna.

Operasi yang dilakukan pada array:

* Penyimpanan data menggunakan `append()`
* Akses data menggunakan index
* Iterasi menggunakan perulangan untuk proses perhitungan
* Fungsi bawaan seperti `max()`, `min()`, dan `sum()`

Penggunaan array mempermudah pengolahan data dalam jumlah banyak secara efisien.

## Analisis Kompleksitas

* Akses elemen array: O(1)
  Mengambil satu data langsung berdasarkan index.

* Perulangan (looping): O(n)
  Digunakan untuk menghitung jumlah mahasiswa lulus.

* Fungsi `max()`, `min()`, `sum()`: O(n)
  Karena harus memeriksa seluruh elemen dalam array.

Kesimpulan analisis:
Sebagian besar operasi bergantung pada jumlah data (n), sehingga kompleksitas didominasi oleh O(n).
